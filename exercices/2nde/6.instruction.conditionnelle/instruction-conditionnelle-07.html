L'instruction conditionnelle en Python

Exercice 7

Complétez le programme suivant en écrivant la fonction maximum() dont les paramètres sont trois nombres a, b, c et qui renvoie le plus grand des ces trois nombres. 


		 
		 
	

	

		

		

	

	
# Solution de l'exercice
def maximum(a, b, c):
    if a > b and a > c:
        return a
    if b > c:           # le maximum est b ou c
        return b
    else:
        return c

x = float( input("1er nombre ?") )
y = float( input("2nd nombre ?") )
z = float( input("3e  nombre ?") )
print("Le plus grand est", maximum(x,y,z))
