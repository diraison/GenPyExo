La boucle for en Python

Exercice 10

Vous devez compléter la fonction compter qui a deux paramètres phrase et lettre et qui renvoie le nombre de fois que le caractère lettre apparait dans la chaîne phrase.

Ainsi print(compter("Coucou Toto","o")) affiche 4, car il y 4 caractères o dans la chaîne "Coucou Toto". 


		 
		 
	

	

		

		

	

	
# Voici une solution

def compter(phrase, lettre):
    n = 0               # initialise le compteur
    for c in phrase:    # pour chaque caractère
        if c == lettre: # s'il y a correspondance
            n = n + 1   # incrémente le compteur
    return n

maphrase = "Coucou Toto"    #? exemple
malettre = "o"              #? exemple

print( compter(maphrase,malettre) )
