Exercice 11

Complétez la fonction supprimerEspaces, de paramètre phrase, qui renvoie une chaîne dans laquelle tous les espaces " " de phrase sont remplacés par des tirets "-".

Ainsi print(supprimerEspaces("Un Deux Trois")) affiche Un-Deux-Trois. 


		 
		 
	

	

		

		

	

	
# Voici une réponse

def supprimerEspaces(phrase):
    s = ""              # resultat à renvoyer
    for c in phrase:    # pour chaque caractère
        if c == " ":    # si c'est un espace
            s = s + "-" # remplacer par un tiret
        else:           # sinon
            s = s + c   # rajouter le caractère
    return s

maphrase = "Un Deux Trois"    #? exemple

print( supprimerEspaces(maphrase) )
