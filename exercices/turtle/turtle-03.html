Exercice 3

Modifiez le programme pour placer dans le repère : le point de coordonnées (-4;5), le point d'ordonnée -3 et d'abscisse 8, et enfin leur milieu.
Vous utiliserez les fonctions suivantes du module turtle :
  • goto(3,2) pour déplacer le crayon aux coordonnées (3;2)
  • dot() pour tracer un point à la position courante du crayon


		 
		 
	

	

		

		

	

	
# Voici une réponse

from turtle import *

fenetre = Screen()
fenetre.setworldcoordinates(-10,-8,10,8)
fenetre.bgpic("repereC2")
hideturtle()
penup()

goto(-4,5)         # premier point
dot()
goto(8,-3)         # second point
dot()
xI = (-4 + 8) / 2     # abscisse du milieu
yI = (5 + (-3)) / 2   # ordonnée du milieu
goto(xI,yI)
dot()
