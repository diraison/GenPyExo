Instruction conditionnelle en Python

Exercice 3

Complétez le programme pour qu'il affiche le résultat de :
  • \(3x-7\)   pour   \(x \geqslant 5\)
  • \(2x-1\)   pour   \(x < 5\)


		 
		 
	

	

		

		

	

	
# Voici une réponse possible

x = float( input("x ? ") )

if x >= 5:
    y = 3 * x - 7
else:
    y = 2 * x - 1

print("y =", y)
