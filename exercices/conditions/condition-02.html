Instruction conditionnelle en Python

Exercice 2

Corriger le programme suivant pour qu'une réduction de 5 % soit appliquée à un prix atteignant ou dépassant les 100 €. 


		 
		 
	

	

		

		

	

	
# Voici une réponse possible

prix = float( input("prix ? ") )

if prix >= 100:
    prix = prix - 5 / 100 * prix

print("prix réduit :", prix)
