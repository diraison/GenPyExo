Exercice 8

Vous devez compléter la fonction frequenceSix(n) pour qu'elle renvoie la fréquence de 6 obtenus lors de n lancers d'un dé.
Elle s'appuiera sur la fonction lancerDe() qui simule le lancer d'un dé, en retournant l'issue obtenue (un nombre entier de 1 à 6).

Remarque : lancerDe() utilise la fonction randint() du module random qui permet de générer des nombres entiers aléatoires (ici entre 1 et 6 inclus). 


		 
		 
	

	

		

		

	

	
# Voici une solution
from random import *

#? simule le lancer d'un dé
def lancerDe(): return randint(1,6)

#  frequenceSix(n) effectue n lancers en
#  comptabilisant le nombre de 6 obtenus
#  et renvoie la fréquence de 6 à la fin
def frequenceSix(n):
    nbsix = 0                  # nombre de 6
    for i in range(n):
        if lancerDe() == 6:    # est-ce un 6 ?
            nbsix = nbsix + 1  # augmente de 1
    return nbsix / n           # frequence

print("Pour 100 lancers la fréquence de 6" \
      " est", frequenceSix(100))
