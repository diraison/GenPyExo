Exercice 5

Modifiez le programme pour obtenir une spirale carrée faisant 6 tours complets espacés de 5 pixels.

Ce programme utilise le module turtle (tortue) qui permet de tracer des figures géométriques à l'aide de fonctions spécialisées :
  • forward(50) pour avancer de 50 pixels
  • left(60) pour tourner à gauche de 60°


		 
		 
	

	

		

		

	

	
# Voici la réponse

from turtle import *  # pour utiliser turtle

color("blue")
c = 5                 # côté initial
for n in range(24):   # 24 quarts de tours
    forward(c)        # avance de c pixels
    left(90)          # tourne à gauche de 90°
    c = c + 5         # augmente le côté de 5
