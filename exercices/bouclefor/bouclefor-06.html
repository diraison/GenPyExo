Exercice 6

Complétez la fonction tracerCarre() ayant trois paramètres : x, y les coordonnées du sommet inférieur gauche et c le côté du carré à tracer.
Vous utiliserez les fonctions suivantes du module turtle :
  • up() pour lever le crayon et ne rien tracer
  • down() pour baisser le crayon et recommencer à tracer
  • goto(-30,70) pour déplacer le crayon aux coordonnées (-30,70)
  • forward(50) pour avancer de 50 pixels
  • left(60) pour tourner à gauche de 60°


		 
		 
	

	

		

		

	

	
# Voici la réponse

from turtle import *

def tracerCarre(x,y,c):
    up()                 # lever le crayon
    goto(x,y)            # se déplacer en (x,y)
    down()               # abaisser le crayon
    for n in range(4):   # 4 côtés
        forward(c)       # côté de longueur c
        left(90)         # angle droit

tracerCarre(-40,20,80)   #? exemple d'appel
